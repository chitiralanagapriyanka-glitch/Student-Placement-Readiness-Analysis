# Student-Placement-Readiness-Analysis
Analysis of student placement readiness using academic, skill and training related data.
This project focuses on analyzing student placement readiness based on different factors like academic performance, skills, certifications, internships and training activities.

## Dataset
The dataset contains student related information required for placement readiness analysis.
